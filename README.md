# Lasso Regression with Feature Selection

## 📌 Overview
This project demonstrates how Lasso Regression performs automatic feature selection by shrinking some coefficients to zero.

## ⚙️ Techniques Used
- Linear Regression
- Lasso Regression
- GridSearchCV (Hyperparameter tuning)
- Feature Scaling

## 🎯 Key Findings
- Lasso removed irrelevant features
- Reduced model complexity
- Maintained or improved performance

## 🧠 Learnings
- L1 regularization performs feature selection
- Alpha controls sparsity
- Simpler models can generalize better
